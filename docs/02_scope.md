# 02 — Project Scope

## In Scope
- CNN genre classifier trained on GTZAN dataset
- Mel-spectrogram preprocessing pipeline
- Genre to mood mapping (4 states: happy, sad, calm, energetic)
- Depression early-warning tracker (7-day genre shift monitor)
- City noise wellness map (simulated with campus recordings)
- Gradio web demo deployed on Hugging Face Spaces
- Accuracy report, confusion matrix, loss curves

## Out of Scope
- Real-time biosensor / wearable hardware integration
- Clinical trials with actual patients
- Mobile app (iOS or Android)
- Paid streaming API (Spotify, YouTube)
- Multi-language lyric analysis
- Multi-city live deployment

## Future Work
All out-of-scope items above are planned as 
Phase 2 extensions after submission.
