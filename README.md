# HealSound
### AI-Powered Music Genre Classification for Health and Social Wellness


---

## What is HealSound?

HealSound is an AI system that classifies music genres using a 
Convolutional Neural Network (CNN) and applies that classification 
to two real-world missions:

- **Health side** — detect early signs of depression through 
  genre listening patterns and recommend therapeutic music
- **Social side** — classify urban soundscapes to build a 
  live city noise wellness map
